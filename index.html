Résumé des differents étapes du Projet du mois de juillet

Projet

Il nous est demandé de développer une solution pour un restaurant

Installation

Pour l'Installation nous avons 4 étape

  • Mise à jour du système
  • Installer les dépendences d'asterisk
  • Télécharger de l'archive d'asterisk
  • Construire et Installer asterisk

Configuration

Nous allons commencé par déplacé les fichiers sip.con, extensions.conf,Voicemail.conf, queues.conf respectivement vers sip.conf.o, extensions.conf.o, Voicemail.conf.o et queues.conf.o

$ sudo mv sip.conf sip.conf.o
$ sudo mv extensions.conf extensions.conf.o
$ sudo mv Voicemail.conf Voicemail.conf.o
$ sudo mv queues.conf queues.conf.o

Creaction des utilisateurs

La configuration des utilisateur est contenue dans le fichier sip.conf

Télécharger la configuration du fichier sip.conf

Voicemail

La configuration du Voicemail est contenue dans le fichier Voicemail.conf. Il s’agit donc de leur messagerie vocale.

Télécharger la configuration du fichier voicemail.conf

Queues

La configuration de Queues est contenue dans le fichier queues.conf. Les files d’attente peuvent être utiles pour gérer les appels entrants

Télécharger la configuration du fichier queues.conf

DialPlant

La configuration du DialPlan est contenue dans le fichier extensions.conf. Le DialPlan est ce qui va définir le routage des appels à travers le serveur
